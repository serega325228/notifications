# Notification Service

Backend-сервис для создания и отправки уведомлений.  
Проект разработан в учебных целях.

## Возможности
- Создание и хранение уведомлений
- Управление статусом уведомлений
- Подготовка к интеграции с различными каналами доставки (email / push / telegram и т.п.)
- Тестируемая и расширяемая архитектура

---

## Технологии
- Python 3.10+
- FastAPI
- Docker / Docker Compose
- Pytest
- SQLAlchemy
- PostgreSQL

---

## Запуск проекта
docker compose up --build

## Тестирование
pytest
